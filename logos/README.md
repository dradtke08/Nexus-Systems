## ğŸ¨ Usage Guidelines Summary

### Background Selection:
- **White/Light Gray:** Use Full-Color logo
- **Navy/Dark:** Use White logo
- **Formal Documents:** Use Navy logo
- **B&W Printing:** Use Grayscale logo

### Size Selection:
- **32-64px:** Icon-only variants
- **256px:** Email signatures, small web elements
- **512px:** Document headers, standard web
- **1024px:** Presentations, high-res web
- **2048-4096px:** Print materials, signage

### Minimum Sizes:
- **Full logo:** 150px width (digital), 1.5" (print)
- **Icon-only:** 32px width (digital), 0.25" (print)

### Clear Space:
Maintain **0.5x icon height** clear space on all sides
